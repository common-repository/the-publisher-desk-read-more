=== Description ===

Easy-To-Use "Read More" Block for Embedded Post Links.

Provides a convenient Gutenberg block that allows an editor to search for an existing post and embed it as a "Read More" link. Especially useful for connecting related content on a site.

=== Functionality ===

Select the "Read More" block through the post editor to open the search panel. Text-based search is available via the input at the top, and after the selection is made the link will appear within the post being currently edited.
